# To-Do App (Vanilla JS)

A simple, lightweight To-Do application built with **HTML, CSS, and vanilla JavaScript**.  
It allows users to add tasks, mark them as completed, delete them, and persist data using `localStorage`.

---

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using browser `localStorage`
- Clean and minimal UI
- No external libraries or frameworks
